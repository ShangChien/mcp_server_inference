# Project-init
```sh
git clone https://github.com/ShangChien/mcp_server_inference.git
pip install uv
```

# requirements install
```sh
uv sync
```

# environment activate
```sh
source ./venv/bin/activate
```

# Run
```sh
./run.sh
```
